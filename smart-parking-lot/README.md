# Smart Parking Lot System

## Project Description
Smart Parking Lot System is a web-based application that assists in efficiently managing and assigning parking slots. The system automatically assigns the closest available slot according to the vehicle's needs, such as EV charging and covered parking.

The project is developed using React and Tailwind CSS and hosted on Vercel.

---

## Live Demo
https://smart-parking-system-psi-two.vercel.app/

---

## Project Features

- Add parking slots
- Display all parking slots
- Automatically park vehicle according to needs
- Remove vehicle from parking slot
- Real-time status update of parking slots
- Attractive and responsive UI

---

## Technologies Used

- React.js
- Tailwind CSS
- JavaScript
- Vite
- Vercel (Hosting)

---

## System Explanation

Each parking slot contains:

- Slot number
- Whether it is covered or not
- Availability of EV charging
- Status of whether it is occupied or available

When a vehicle arrives, the system automatically assigns the closest available slot according to the vehicle's needs.
